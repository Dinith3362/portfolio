Smart Shopping Cart System

An IoT-enabled shopping cart using UHF RFID, ESP32, and Firebase for automated checkout.

Technical Details

This embedded system adopts a three-tier, client-server architecture with IoT-enabled shopping carts. Used UHF RFID tags and reader, ESP32 microcontroller, Server controller lock, and Tab. The cart devices connect directly to Firebase over Wi-Fi using Firebase's real-time database and cloud functions. Otherwise this embedded system used QR-based payments, and security verification to automate checkout.

Research Experience

2026 - Present

Adaptive Robust Steganography Resilient to Aggressive JPEG Compression and Geometric Scaling

Addressing the "robustness gap" in digital image steganography on social media platforms.

Advanced Research Analysis

Research looks at digital image steganography and highlights a common problem known as the "robustness gap," where hidden data gets damaged or completely lost because of heavy compression and resizing on social media platforms. To tackle this issue, this review modern techniques and explains why a content-adaptive approach works best. It uses methods like DWT, SVD, and Reed-Solomon error correction to make the hidden communication more reliable and resistant to distortion.
